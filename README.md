## Project setup

In the project directory, you can run:

```
npm install
```

### To begin development, run

```
npm start
```

### To begin server.js, run

```
nodemon server.js
```

Open [http://localhost:8081](http://localhost:8081) to view it in the browser.

The page will reload if you make edits.
